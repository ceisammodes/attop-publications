# XYZ File Deposit

The **initial XYZ geometry files**, which are too large for GitLab, are available for each ATTOP publication in a **public Uncloud space** dedicated to the ATTOP project.
These files are referenced in our **ATTOP publications GitLab repository** in the [`xyz_file_deposit.md`](https://gitlab.univ-nantes.fr/modes/public/attop/publications-attop) file.

Each publication that uses this XYZ data has its own **DOI-named directory**, accompanied by a **README.md file** that explains the data.

**Example:**
In the [publications repository](https://gitlab.univ-nantes.fr/modes/public/attop/publications-attop), the data for the publication *“Ultrafast probing of isotope-induced explicit symmetry breaking in ethylene”* is referenced in the directory `doi.org--10.1038--s42004-025-01621-z`. This directory contains a **README.md file** titled with the publication name, providing a simple explanation of the files, their structure, and their description.
